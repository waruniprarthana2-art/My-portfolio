Waruni Prarthana — Personal Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript.

🔗 Live Demo
[View Website](https://your-github-pages-link.github.io)

 📌 Features
- Responsive design for all screen sizes
- Animated skill bars
- Project showcase with modal popups
- Blog section with Swiper slider
- Contact form with EmailJS integration
- About page with Education, Hobbies & Favourites

 🛠️ Built With
- HTML5
- CSS3
- JavaScript
- [Swiper.js](https://swiperjs.com/) — Blog slider
- [Typed.js](https://github.com/mattboldt/typed.js/) — Typing animation
- [EmailJS](https://www.emailjs.com/) — Contact form
- [Boxicons](https://boxicons.com/) — Icons

 📁 Pages
- `index.html` — Main portfolio page
- `projectsa.html` — Projects page
- `about.html` — About page

 🚀 How to Run
1. Clone the repository
```bash
   git clone https://github.com/waruniprarthana2-art/your-repo-name.git
```
2. Open `index.html` in your browser
